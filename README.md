## Zombie Arena

### Controls:
- Esc - quit the game
- Enter - start/pause/continue the game
- W, A, S, D - control the player
- R - reload the gun
- Left mouse click - shoot the bullet
- 1, 2, 3, 4, 5, 6 - pick one of the 6 upgrades
